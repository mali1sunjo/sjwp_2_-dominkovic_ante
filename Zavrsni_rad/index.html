Početna lekcija

Edukativna web stranica za učenje HTML-a i CSS-a

Ova stranica služi kao mali online tečaj za početnike. Cilj je da korisnik korak po korak nauči što je HTML, kako CSS mijenja izgled stranice, kako nastaje layout, te kako se od osnovnih elemenata dolazi do gotovog mini projekta.

Kreni s HTML osnovama Isprobaj live editor

Kako koristiti stranicu

  • Stranice čitaj redom jer gradivo ide od jednostavnog prema složenijem.
  • Svaki primjer sadrži kod, prikaz rezultata i kratko objašnjenje.
  • Na kraju svake cjeline nalazi se savjet, česta pogreška i mala vježba.
  • Na stranici Primjeri nalazi se pravi editor za isprobavanje koda.

Što je HTML

HTML gradi strukturu stranice: naslove, tekst, linkove, slike, tablice i forme.

Što je CSS

CSS definira boje, tipografiju, razmake, rasporede i ukupni izgled stranice.

Što ćeš napraviti

Na kraju vodiča složit ćeš mali portfolio projekt i razumjeti kako je napravljen.

Plan učenja

Prvo savladaj HTML osnove, zatim semantičke elemente, nakon toga CSS i layout. Tek tada prijeđi na gotove primjere i mini projekt.

Zašto je ovo korisno

Umjesto da samo čitaš teoriju, ovdje možeš odmah gledati konkretan kod i rezultat. Takav način učenja čini gradivo preglednijim i lakšim za pamćenje.

Kako HTML i CSS rade zajedno

HTML određuje sadržaj, a CSS odlučuje kako će taj sadržaj izgledati.

Savjet: Uči tako da za svaki primjer prvo pogledaš strukturu HTML-a, a tek onda što CSS vizualno mijenja.

Česta pogreška: Početnici često uče CSS prije nego razumiju HTML strukturu. Tada styling djeluje zbunjujuće.

Vježba: U primjeru promijeni tekst gumba, zatim promijeni boju naslova i provjeri znaš li objasniti što radi svaki red koda.
