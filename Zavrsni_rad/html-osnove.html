Početnički HTML

HTML osnove

HTML je temelj svake web stranice. Njegova glavna uloga je dati sadržaju strukturu: naslov, odlomak, listu, poveznicu, sliku, tablicu ili formu. U ovoj lekciji krećemo od najvažnijih početničkih elemenata.

Što trebaš zapamtiti

  • HTML opisuje sadržaj, ne dizajn.
  • Većina elemenata ima početnu i završnu oznaku.
  • Atributi daju dodatne informacije elementima.
  • Dobra struktura olakšava CSS i pristupačnost.
