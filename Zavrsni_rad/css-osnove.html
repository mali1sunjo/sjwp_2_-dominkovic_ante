CSS osnove

CSS je zadužen za izgled web stranice. Njime mijenjaš boje, veličine, fontove, razmake, obrube, hover efekte i puno više. Dobar CSS ne samo da uljepšava stranicu, nego i čini sadržaj preglednijim.

Što trebaš znati

  • pravilo = selektor + deklaracije
  • svojstvo određuje što mijenjaš
  • vrijednost određuje kako izgleda
  • CSS djeluje na HTML elemente

Selektori, boje i tipografija

Prvo nauči kako “uhvatiti” element i promijeniti mu izgled.

Temeljni alati

Primjer osnovnog selektora

Primjer 1

Ovim pravilom ciljaš paragraf unutar određenog bloka i mijenjaš mu boju i debljinu.

Kod 
.selector-demo p {
  color: #0f766e;
  font-weight: 700;
}
Rezultat

Ovaj tekst je odabran selektorom i izgleda naglašenije.

Primjer boje i pozadine

Primjer 2

CSS lako stvara vizualni identitet stranice pomoću boje teksta, pozadine i zaobljenih rubova.

Kod 
.color-demo {
  background: linear-gradient(135deg, #0f766e, #2563eb);
  color: white;
  padding: 1rem;
  border-radius: 16px;
}
Rezultat
Ovaj blok ima gradijentnu pozadinu i svijetli tekst.

Primjer tipografije

Primjer 3

Dobro uređena tipografija čini sadržaj profesionalnijim i lakšim za čitanje.

Kod 
.typography-demo h3 {
  font-size: 1.6rem;
  letter-spacing: -0.02em;
}

.typography-demo p {
  line-height: 1.7;
}
Rezultat

Naslov s boljom hijerarhijom

Paragraf s dovoljno prostora između redaka mnogo je ugodniji za čitanje od zbijenog teksta.

Savjet: Uzmi 2–3 glavne boje i ponavljaj ih dosljedno kroz cijelu stranicu.

Česta pogreška: Nemoj koristiti previše različitih fontova i boja na istoj stranici.

Vježba: Napravi blok s naslovom, paragrafom i gradijentnom pozadinom po uzoru na gornji primjer.

Box model, interakcije i varijable

Kad savladaš razmake i hover efekte, dizajn izgleda puno profesionalnije.

Napredniji temelj

Primjer box modela

Primjer 4

Svaki element u CSS-u ponaša se kao kutija: sadržaj, padding, border i margin.

Kod 
.demo-box {
  padding: 1rem 1.25rem;
  border: 4px solid #0f766e;
  margin-top: 0.75rem;
}
Rezultat

Vanjski tekst

Ovo je sadržaj unutar box model kutije.

Primjer hover efekta i CSS varijable

Primjer 5

Varijable čine dizajn dosljednim, a hover daje osjećaj interakcije.

Kod 
:root {
  --glavna-boja: #0f766e;
}

.gumb {
  background: var(--glavna-boja);
  color: white;
  border-radius: 12px;
}

.gumb:hover {
  transform: translateY(-2px);
}
Rezultat

Pređi mišem preko gumba:

Savjet: Kad god više puta koristiš istu boju, smisli CSS varijablu. Tako se dizajn lakše održava.

Česta pogreška: Početnici često zaborave da padding povećava unutarnji razmak, a margin vanjski.

Vježba: Napravi karticu koja ima obrub, padding, pozadinu i gumb s hover efektom.

Sljedeći korak: otvori CSS layout i nauči rasporediti sadržaj pomoću Flexboxa i Grida.
