Flexbox i Grid

CSS layout

Layout je način na koji sadržaj raspoređuješ po stranici. Najvažniji moderni alati su Flexbox i Grid. Flexbox je odličan za jedan red ili stupac elemenata, a Grid za složenije mreže i više dimenzija.

Kad koristiti što

  • Flexbox: nav, gumbi, redovi kartica, poravnanje
  • Grid: galerije, dashboard, mreža kartica, glavni raspored
  • Media query: prilagodba širini zaslona

Flexbox u praksi

Flexbox je najčešći izbor za navigacije, gumbe i kartice koje se trebaju uredno poravnati.

Jedna dimenzija

Primjer horizontalne navigacije

Primjer 1

Jednim pravilom dobivaš uredan red elemenata i lakše poravnanje po osi.

Kod 
.menu {
  display: flex;
  gap: 12px;
  align-items: center;
}

Primjer kartica koje se prelamaju

Primjer 2

Kad nema dovoljno mjesta, elementi mogu prijeći u novi red pomoću flex-wrap.

Kod 
.kartice {
  display: flex;
  flex-wrap: wrap;
  gap: 12px;
}

.kartice article {
  flex: 1 1 140px;
}

Flexbox
Grid

Savjet: Kad želiš poravnanje u jednom smjeru, prvo pomisli na Flexbox.

Česta pogreška: Početnici često pokušaju napraviti cijeli kompleksan layout samo Flexboxom, iako bi Grid bio čistije rješenje.

Vježba: Napravi red od tri gumba koji se na manjem ekranu prelamaju ispod jedan drugog.

Grid i responzivni raspored

Grid je idealan za galerije, sekcije s više kartica i dvostupčane rasporede.

Dvije dimenzije

Primjer Grid mreže

Primjer 3

Grid stvara mrežu redaka i stupaca, pa je izvrstan za složenije rasporede.

Kod 
.mreza {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 12px;
}

Primjer responzivnog rasporeda

Primjer 4

Media query mijenja raspored na većim širinama zaslona. To je bit mobile-first pristupa.

Kod 
.layout {
  display: grid;
  gap: 12px;
}

@media (min-width: 760px) {
  .layout {
    grid-template-columns: 2fr 1fr;
  }
}
Rezultat
Glavni sadržaj
Sidebar

Savjet: Kreni od jednostavnog rasporeda za mobitel i tek onda proširi layout za veće ekrane.

Česta pogreška: Nemoj dizajnirati samo za desktop. Uvijek pogledaj kako stranica izgleda na užoj širini.

Vježba: Napravi mrežu od četiri kartice koja na mobitelu ima jedan stupac, a na većem ekranu dva ili tri.
